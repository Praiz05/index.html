## Brief Reflection

### What I Learned
In this lab, I learned how to create modern layouts using CSS Grid and Flexbox. I converted a flex layout to a grid layout with `grid-template-columns`, which gave me more control over the arrangement of items. I also learned how to create responsive designs using media queries that adapt to different screen sizes, changing both layout and colors for mobile devices. Additionally, I practiced creating interactive elements with hover effects and building a functional navigation bar using Flexbox.

### Challenges Faced
The main challenge was understanding the difference between Flexbox (one-dimensional layout) and Grid (two-dimensional layout). Initially, I had issues with the grid responsiveness, but after experimenting with `grid-template-columns: repeat(3, 1fr)` and using media queries to change it to `1fr` on mobile, everything worked smoothly. Testing across different screen sizes using browser developer tools helped me ensure the layout worked properly on both desktop and mobile devices.
