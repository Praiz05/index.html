## Brief Reflection

### What I Learned
In this lab, I learned how to apply CSS using all three methods: inline CSS (directly in HTML tags), internal CSS (using `<style>` tag in the head), and external CSS (linking a separate .css file). I also learned how to style elements with borders, padding, and margins, and how to create gradient backgrounds for visual appeal.

### Challenges Faced
The main challenge was understanding how different CSS methods override each other. Inline CSS has the highest specificity, followed by internal, then external. I also had to ensure the gradient background worked correctly across different browsers. Using browser developer tools helped me experiment with styles temporarily to see what worked best.
