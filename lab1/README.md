## Brief Reflection

### What I Learned
Through this lab, I learned how to use semantic HTML tags like `<header>`, `<nav>`, `<article>`, and `<footer>` to make web pages more accessible and structured. I also learned how to create tables with multiple rows of data, and how to build forms with various input elements including radio buttons, checkboxes, and textareas.

### Challenges Faced
The main challenge was ensuring all HTML elements were properly nested and validated. I initially had some issues with form structure and making sure all form elements had properly associated labels. Using the W3C HTML validator helped me identify and fix these issues.
