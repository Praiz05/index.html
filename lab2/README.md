## Brief Reflection

### What I Learned
In this lab, I learned how to use semantic HTML tags like `<header>`, `<nav>`, `<section>`, `<article>`, and `<footer>` to structure a page clearly. I also practiced adding images with proper `alt` text, building tables with multiple rows, and creating forms with text inputs, radio buttons, checkboxes, and a textarea.

### Challenges Faced
The main challenge was keeping all form elements properly labeled and ensuring the table structure was correct. I also had to be careful with semantic tag placement so the page layout stayed logical and valid.
