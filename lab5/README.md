## Brief Reflection

### What I Learned
In this lab, I learned the fundamentals of JavaScript including variables, functions, control structures, and interacting with users through console output, alerts, and prompts. I learned how to declare variables with `let`, create functions with parameters and return values, use conditional statements (`if-else`), and loop through arrays. I also learned how to move JavaScript code from inline `<script>` tags to an external `.js` file, which is better for code organization and reusability.

### Challenges Faced
The main challenge was understanding how to handle user input properly, especially converting string inputs from `prompt()` to numbers for mathematical operations. I initially forgot to use `Number()` to convert the input, which caused string concatenation instead of addition. I also had to add error handling to check if the user entered valid numbers or cancelled the prompt. Another challenge was ensuring all activities were completed in the correct order and that the external script loaded properly using the `defer` attribute.
